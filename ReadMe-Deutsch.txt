Beschreibung:
-------------
  Living Code simuliert künstliches Leben.
  Sie können selbst Lebewesen entwerfen oder einfach die Entwicklung der Lebewesen beobachten.
  Es besteht die Möglichkeit die Entwicklung der Lebewesen Kontrollieren: 
  Sie können ihnen Nahrung geben, Sie mit Viren oder DNA infizieren, das Aussehen ändern oder auch entfernen. 

  Ihre Änderungen wirken sich auf die Entwicklung der Kreaturen aus. 


Compiling:
----------
  Windows: Erstellen Sie eine threadsafe executable.
  Linux: Erstellen Sie KEIN threadsafe executable.

Steuerung:
----------
  Mit dem Menu Insert können sie verschiedene Lebewesen einfügen.
  Die Lebenwesen werden im Unterodner Insert gepeichert, in dem Sie auch eine Ordnerstukturen und Lebenwesen anlegen können.
  Wenn Sie im Menü eine Kreatur anklicken, werden die Enstellungen der Kreatur angezeigt. Über die Schaltfläche "New" können Sie die Kreatur in die Simulation einfügen.  
  Mehr Informationen zur Steuererung und weitere Erklärungen zu den Befehlen finden sie unter in der Beiliegenden Hilfe im Unterverzeichnis "Help\Hilfe.html" oder über den Menüpunkt "German Help" im Menü "?".


Bekannte Fehler:
----------------
  Das Programm kann für einen Längeren Zeitraum nicht reagieren, wenn es sich für längere Zeit im Hintergrund befunden hat.
  In diesem Fall müssen Sie warten, bis das Programm wieder läuft (Dies kann manchmal ein paar Minuten dauern).
